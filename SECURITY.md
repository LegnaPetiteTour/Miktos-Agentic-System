# Security Policy

## Supported Versions

This project is in active early development. Security fixes are applied to the
latest state of main.

## Reporting a Vulnerability

Please do not report security vulnerabilities publicly.

Report privately by contacting the maintainers through GitHub security
advisories for this repository.

Include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Impact assessment
- Suggested remediation if available

We will acknowledge receipt within 3 business days and provide status updates
as triage progresses.
